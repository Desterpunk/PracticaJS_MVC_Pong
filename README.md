Cual es la diferencia entre == y ===?

Los 3 signos iguales significan "igualdad sin coerción de tipo". Utilizando el triple igual, los valores también deben ser iguales en tipo.
0 == false   // true
0 === false  // false
1 == "1"     // true
1 === "1"    // false
null == undefined // true
null === undefined // false
'0' == false // true
'0' === false // false
